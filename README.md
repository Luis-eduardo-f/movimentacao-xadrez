# Movimentação de Peças de Xadrez

Trabalho desenvolvido para a disciplina de Introdução à Programação de Computadores.

## Objetivo
Implementar, em Python, a validação da movimentação das peças de xadrez,
de acordo com suas regras básicas.

## Peças implementadas
- Torre
- Bispo
- Cavalo
- Rainha
- Rei
- Peão

## Execução
```bash
python xadrez.py
